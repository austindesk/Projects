Project Overview:
This Weather App allows users to check the current weather conditions for any city worldwide. Users can enter the city name and country code to get temperature, weather descriptions, and other details. The app fetches real-time data from the OpenWeather API.

Features:
Search by city and country code
Display temperature and weather descriptions
Clean, user-friendly interface with a logo

Setup Instructions:
Clone the repository.
Add your OpenWeather API key to script.js.
Place your logo image in the /images directory.
Open index.html in your browser.


Technologies Used:
HTML, CSS, JavaScript
OpenWeather API
